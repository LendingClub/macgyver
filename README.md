## Building From Source

You must have Neo4j running on your local machine.

To Build and Test:

```bash
./gradlew clean check
```

To run:
```bash
./gradlew run
````

Point your web browser at [http://localhost:8080](http://localhost:8080) and log in with admin/admin.
